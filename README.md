# PGAdaptor

A simple PG handler. Translates Structs into PG and back.

## Installation

Add this line to your application's Gemfile:

    gem 'pg_adaptor'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pg_adaptor
